#  Sooriya Mushrooms Website

A modern and responsive business website developed for **Sooriya Mushrooms**, a mushroom cultivation and food products business in Sri Lanka.

This website provides customers with information about the company, mushroom products, nutrition benefits, and an easy way to contact and place orders.

---

##  About The Project

Sooriya Mushrooms is a web-based platform designed to showcase the company's mushroom products and services.

The main goal of this project is to create a simple, user-friendly, and responsive website that helps customers learn about mushroom products and connect with the business easily.

---

##  Features

###  Home Page
- Attractive hero section with mushroom images
- Introduction to Sooriya Mushrooms
- Featured products
- Customer-focused design

###  About Us
- Company introduction
- Business background
- Vision and mission information

###  Products Page
- Display available mushroom products
- Product descriptions
- Order through WhatsApp integration

###  Company Profile
- Downloadable company profile PDF

###  Nutrition & Tips
- Mushroom nutrition information
- Health benefits
- Cooking and storage tips

###  Contact Page
- Contact details
- Location map integration
- Customer inquiry form
